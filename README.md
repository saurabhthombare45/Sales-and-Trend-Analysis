# Sales-and-Trend-Analysis
Data Analyst project showing Sales and Trend using Python , SQL and POWER BI

This project demonstrates a complete data analysis workflow starting from raw data to business insights and dashboard visualization. The objective is to analyze the dataset, extract meaningful insights, and present findings using SQL and Power BI.

# The project includes:

Data Loading

Data Cleaning

Feature Engineering

Exploratory Data Analysis (EDA)

SQL and MySQL Queries

Power BI Dashboard

# Tools and Technologies

Python

Pandas

NumPy

Matplotlib

Seaborn

SQL

# Project Workflow
1. Data Loading

Imported the dataset using Pandas.

Reviewed structure using head(), info(), and describe() functions.

2. Data Cleaning

Handled missing values.

Removed duplicate records.

Corrected inconsistent data types.

Renamed columns where necessary.

3. Feature Engineering

Created new calculated columns.

Generated business metrics.

Performed categorical encoding where required.

4. Exploratory Data Analysis

Analyzed distributions of numerical variables.

Performed category-wise and location-wise analysis.

Identified trends and patterns.

Conducted correlation analysis.

5. SQL and MySQL Analysis

Created a database in MySQL.

Imported the dataset into the database.

Performed:

Aggregations (SUM, AVG, COUNT)

GROUP BY operations

JOIN queries

Subqueries

Window functions

Ranking analysis

6. Power BI Dashboard

Created an interactive dashboard.

Designed KPI cards for key metrics.

Built charts for category, location, and time-based analysis.

Added filters and slicers for better interactivity.
